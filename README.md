# NovaClean Riviera

Site vitrine premium pour NovaClean Riviera.

## Installation

```bash
npm install
npm run dev
```

Ouvrir `http://localhost:3000`.

## Déploiement Vercel

1. Créer un repository GitHub.
2. Upload/push ce dossier.
3. Importer le repository dans Vercel.
4. Vercel détectera automatiquement Next.js.

# NovaClean Riviera

Site vitrine Next.js prêt pour GitHub et Vercel.

## Lancer en local

```bash
npm install
npm run dev
```

## Déployer sur Vercel

1. Créer un repo GitHub.
2. Upload tous les fichiers de ce dossier.
3. Importer le repo dans Vercel.
4. Laisser Vercel détecter Next.js.
5. Deploy.
